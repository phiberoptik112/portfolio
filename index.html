Acoustics ASTM Test Report Generator

I created a Python GUI and processing script backend that generates ASTM test reports from ASTC, AIIC, NIC, and DTC acoustics tests. The script processes sound level data combines them with the test environment aspects to generate a report with the results of the test. This is replacing a manual entry excel spreadsheet and reduces the test data processing and report generation and formatting time from hours to minutes.

Airpods Max: Pressure Relief Vent Production Line Tester

During the hardware development phase of Airpods Max, the primary pressure relief vent for the speaker front-volume was shown to produce a tone when airflow was introduced. I was tasked with finding the root cause of this issue and creating a production line tester to validate the solution. This was a critical project for the Airpods Max launch and I was able to find the root cause and create a solution that was implemented in the production line to validate the solution.

The fix primarily consisted of developing a frequency analysis technique of Q-factor measurement of a tone produced when air was forced through the pressure relief vent.
